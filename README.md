### Feature
- **Canvas渲染**
- **编写组件语法，易于上手**
- **响应式动态更新**
- **自动处理连接线，指针**
- **多种内置图形（Rect, Circle, Isogon, Line, Curve, PolyLine, Arrow, Node...）**
- **自定义复合图形（Composite）**
- **丰富的配置项，可随意定制可视化视图**
- **多种锚点模式：内置锚点，自定义锚点，甚至动态锚点**
